# Logo Setup Instructions

## Adding the Logo

Please save the metallic triangle logo image as:
```
web/public/images/logo.png
```

The logo should be:
- A transparent PNG file (or with black background removed)
- The metallic/chrome triangle design provided
- Recommended size: 512x512px or similar square dimensions

## Current Implementation

The logo is used in:
- `web/index.html` - Browser tab favicon
- `web/src/components/site/Logo.tsx` - Main logo component in navbar
- Displayed in the navbar across all pages
- Has hover effects and transitions
- Shows as the website icon in browser tabs

## File Location

Save your logo file to: `E:\PROJECTS\verity-AI(Blockchain)\verity-ai-main\web\public\images\logo.png`

## How to Save the Logo

1. Right-click on the logo image in the chat
2. Select "Save image as..."
3. Navigate to: `E:\PROJECTS\verity-AI(Blockchain)\verity-ai-main\web\public\images\`
4. Save it as `logo.png`
5. Refresh your browser to see the new favicon in the tab
